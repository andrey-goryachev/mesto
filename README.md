# Проект: Место
Проект сделан в рамках курса Яндекс Практикум, реализовано:
- адаптивная верстка
- всплывающее модальное окно, введенная информация сохраняется на странице

Технологии:
- HTML
- CSS
- JavaScript
- Grid
- Flex
